Les 2

In de eerste les hebben we de opdrachten uit het boekje gemaakt die zich bevinden bij les 1. De bedoeling in les 2 was dan om zelf al de website tot stand te brengen zodat we tijdens de laatste lessen gewoon nog de tekst en fotootjes er moesten in plaatsen.

We waren niet verplicht om naar de les te gaan dus hebben we samen afgesproken op ons kot om deze website al zo goed als mogelijk af te werken. We hebben bij het maken van deze website gebruik gemaakt van de IDE Webstorm. Dit is gratis omdat je als student recht hebt op de Student Pack van Github. Daarbij hebben we de website gehost op Github zelf. We hebben Bootstrap v4.1 gebruikt als front-end framework.
